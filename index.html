Juice CSS Inliner

Juice is a CSS inliner available on npm.

Unlike a lot of other CSS inliners available online, Juice has more options, can be included in your Node.js projects, and works in client JavaScript as shown here. When you install Juice from npm, you can also inline contents from external CSS, images, and JavaScript files. This simple online version expects all CSS to be included within style tags in the Input box.

Enjoy!
